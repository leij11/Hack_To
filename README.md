## backend:
Create a virtual environment and activate it: https://packaging.python.org/guides/installing-using-pip-and-virtual-environments/

Install dependencies:
```
python3 -m pip install -r requirements.txt
```

Run program:
```
python3 opencv.py
```

### Run flask server
MacOs or Linux:
```
python3 main.py
```
Windows:
```
py main.py
```
